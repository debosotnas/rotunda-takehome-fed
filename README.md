# Rotunda Take-home FED

Each result for the three exercises will be found in a separate directory:

- Zoo: [link](https://github.com/debosotnas/rotunda-takehome-fed/tree/main/zoo)
- Alarm: [link](https://github.com/debosotnas/rotunda-takehome-fed/tree/main/alarm)
- UrlParser: [link](https://github.com/debosotnas/rotunda-takehome-fed/tree/main/url-parser)

The solutions are VanillaJS based. Babel was installed only to support Jest for testing.

To run the tests:

    npm install
    npm test
